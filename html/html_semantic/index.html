HTML Semantic Elements 📌

Semantic Elements in HTML basically mean elements with a meaning to the developer and the browser

Example of Semantic elements


    <article>   <section>   <summary>   <time>   <nav>   <mark>
    <aside>   <main>   <header>   <details>   <footer>   <figure>   <caption>

As you can see, there are some semantic elements to define different parts of a webpage

Commonly used Semantic elements

The <header> element

  • The <header> element is an introductory element and used for introduction or navigational links.

The <nav> element

  • The <nav> element contains a set of major navigational links.

The <article> element

  • The <nav> element typically contains blog post or articles.

The <aside> element

  • The <aside> element defines a sidebar or any content placed on a side.

The <section> element

  • The <section> element can be used to define sections.

The <aside> element

  • The <aside> element defines a sidebar or any content placed on a side.

The <footer> element

  • The <footer> element containes the footer of the page.

The <details> element

  • The <details> element allows us to create an element that a user can show or hide.

The <figure> element

  • The <figure> element contains any figure or illustrations.

The <figcaption> element

  • The <figcaptione> element defines a caption for <figure> element.
